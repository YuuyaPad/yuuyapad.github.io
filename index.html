YuuyaPad

YuuyaPad is a notepad written in C# for Windows only, and aims to be multifunctional.
However, it is inferior to Notepad++.

It is open source under the MIT license, so anyone can contribute to YuuyaPad or build it from source code.

OS Requirements

The following operating systems are required to run YuuyaPad.

  • Minimum: Windows 7
  • Recommended: Windows 10 22H2 or Windows 11

Unsupported OS

YuuyaPad does not support the following operating systems.
We will also not port YuuyaPad to these operating systems.

  • macOS
  • Linux
  • ChromeOS
  • iOS
  • Android
  • Windows Server Core

Feature goals

YuuyaPad is being developed with the goal of implementing all of the following features.
For more details and progress, see the README.md in the GitHub repository.

  • Finalize the menu bar
  • Save feature
  • Multilingual Support
  • Dark Mode support
  • Multiple simultaneous editing via tabs or MDI
  • Native support for ARM

The following features are not planned for implementation, but may be added if there are contributors.

  • Support for Windows XP/Vista
  • Extensibility via plugins
